# Retail Sales Analysis Project

##  Project Overview

This project focuses on analyzing retail sales data to understand business performance, customer behavior, and product profitability. The goal is to identify key trends and insights that can help improve sales strategies and decision-making.

---

## Tools & Technologies Used

* **Excel** → Data cleaning, pivot tables, dashboard creation
* **SQL** → Data querying, aggregation, and joins
* **Power BI** → Data modeling, interactive dashboards, visualization

---

## Project Workflow

### 🔹 1. Excel Analysis

* Created pivot tables to analyze:

  * Sales by Region
  * Profit by Category
  * Customer-wise performance
* Built dashboard using:

  * Clustered column and bar charts
  * Slicers for dynamic filtering

---

### 🔹 2. SQL Analysis

* Created tables: `orders`, `customers`, `products`
* Performed queries using:

  * `SELECT`, `WHERE`, `GROUP BY`, `ORDER BY`
* Used **INNER JOIN** on `customer_id` to combine orders with customer data

---

### 🔹 3. Power BI Dashboard

* Built a data model with:

  * **Fact Table** → Orders
  * **Dimension Tables** → Customers, Products
* Created 3 pages:

  * **Overview Dashboard** → Sales and Profit KPIs
  * **Product Analysis** → Product performance
  * **Customer Analysis** → Customer contribution
* Used slicers for interactivity

---

## Key Concepts Applied

* Data Cleaning & Transformation
* Pivot Table Analysis
* SQL Joins & Aggregations
* Data Modeling (Star Schema)
* Dashboard Design

---

## Key Insights

* Certain products generate higher sales and profit
* Some regions contribute more to overall revenue
* A small number of customers contribute significantly to total sales

---

## Business Value

This project demonstrates how data analysis can help businesses identify top-performing products, optimize sales strategies, and improve customer targeting.
